🔌 接入说明（hybrid 怎么上报）

你的 hybrid 应用直接用 HTTP POST 把每个事件写到 Firebase 实时库即可（开放规则下无需鉴权，免 SDK）。 端点（运行时从内置配置解出，下面已填好）：

请求体（JSON）字段：device 必填（设备/安装唯一ID，hybrid 自己生成并持久化）、type 必填（事件类型）；其余可选：name(详情) platform(android/ios/web) ver(版本) level(info/warn/error，默认 info) ts(毫秒时间戳，省略则用服务器时间) data(任意自定义对象)。

{ "device": "dev-abc123", "type": "open", "name": "启动到首页", "platform": "android", "ver": "1.2.3", "level": "info", "ts": 1700000000000 }
